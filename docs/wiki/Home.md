# TranslateR Wiki

TranslateR is a portable gettext `.po` editor for translator handoff workflows.

Use these pages:

- [Translator Guide](Translator-Guide): edit one `.po` file and export a `.tpatch`.
- [Maintainer Guide](Maintainer-Guide): review and merge one or more `.tpatch` files.
- [Release Process](Release-Process): how automatic releases are cut from GitLab CI.
